휴대폰안심(native)

이용 안내

모바일 환경에 최적화 된 휴대폰안심 서비스를 통해 보안 위협으로 부터 내 폰을 안전하게 보호해 보세요.

보안 검사
보안검사를 통해 감지된 위험 알림

단말 보안상태, RAM최적화,
설치된 App의 위험 여부를 확인해 보세요.

네트워크 검사
네트워크 보안 상태 알림

공공, 개인 Wi-Fi 위험도 검사하고
위험에 노출된 Wi-Fi 정보 확인해 보세요

웹 검사
실시간 탐지를 통해 위험한 링크 알림

URL 검사를 통해 링크의 위험 요소 확인하고
세이프브라우징, 스미싱탐지를 통해
위험한 링크를 실시간으로 탐지해 보세요.
(iOS의 경우 스미싱 탐지 미 제공)

보안 사고 예방
사고 후 보상까지 사후 관리 기능 제공

보상보험을 통해 피싱으로 인한 피해 발생 예방하고,
0505 안심번호 발급을 통해
내 휴대폰번호가 노출되지 않게 관리해 보세요.
